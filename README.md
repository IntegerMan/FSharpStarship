# Modeling a Starship in F#
Code for an upcoming talk on Modeling a Starship in F#. Intended as a fun introduction to the F# programming language.

## Decisions

### Visualization

- 2D Tiles
- 2D Rectangles
- Lines / Charts
- Tac Map

### Timing

- Real-time
- Turn-based

## Systems Modeled

### Environmental

- Power Generation
- Power Storage
- Power Consumption
- Heat
- Environment Gas Composition (O2 / CO2)
- Storage Tanks
- Fluid Pipes / Tanks (Water, Untreated Water, Fuel, etc.)
- Water Recycling
- Air Recycling

### Crew

- Crew Locations
- Crew Tasks
- Crew Efficiency
- Crew Fatigue

### Operational

- Storage
- Current Vector
- Thrust
- Fuel
